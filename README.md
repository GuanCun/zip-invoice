# zip-invoice

📦 QQ 邮箱发票压缩包 → 自动整理成公司报销系统可识别格式

一个用于自动化处理 QQ 邮箱发票压缩包的命令行工具。

## 使用场景

在 QQ 邮箱中管理发票，多选下载后会生成一个包含 Excel 发票明细和多个 PDF 发票的压缩包。但公司报销系统只认特定格式的发票文件名。

本工具自动完成：
- ✅ 解压 QQ 邮箱发票压缩包
- ✅ 读取 Excel 发票明细表
- ✅ 根据日期和金额匹配，自动重命名 PDF 文件
- ✅ 批量转换为报销系统要求的命名格式

## 安装

```bash
npm install
```

## 使用方法

### 方式一：打包成可执行文件（推荐）

项目已提供打包好的可执行文件：
- `index-linux`（Linux 系统）
- `index-macos`（macOS 系统）

**使用步骤：**

1. 将 QQ 邮箱下载的发票压缩包放到固定目录
2. 双击对应系统的可执行文件
3. 在终端中选择要处理的压缩包编号
4. 自动完成解压和重命名，即可上传报销系统

### 方式二：直接运行脚本

```bash
node index.js
```

## 输入格式（QQ 邮箱导出）

- **压缩包**：QQ 邮箱多选发票下载生成的 ZIP 文件
- **Excel 文件**：压缩包内的发票明细表（.xlsx），需包含以下列：
  - `开票日期`：发票日期
  - `金额`：发票金额（精确到两位小数）
  - `发票类型`：发票的类型标识
- **PDF 文件**：原始命名格式为 `YYMMDD_金额_其他信息.pdf`

## 输出格式（报销系统要求）

PDF 文件将被重命名为：`YYMMDD-发票类型-金额.pdf`

**示例：**
- 原文件：`240315_1280.50_QQ邮箱.pdf`
- 新文件：`240315-餐饮-1280.50.pdf`

## 工作流程

```
📥 QQ 邮箱下载发票
    ↓
📦 生成压缩包（含 Excel + PDF）
    ↓
🖱️ 双击执行文件 / 运行脚本
    ↓
📂 自动解压到同名文件夹
    ↓
📊 读取 Excel 发票明细
    ↓
🔄 匹配 PDF 文件并重命名
    ↓
✅ 完成！可直接上传报销系统
```


## 依赖

- `unzipper`：解压 ZIP 文件
- `xlsx`：读取 Excel 文件
- `readline`：命令行交互

## 打包可执行文件

如需重新打包，可使用 [pkg](https://github.com/vercel/pkg)：

```bash
npm install -g pkg
pkg index.js --targets node18-linux-x64,node18-macos-x64 --output index
```
